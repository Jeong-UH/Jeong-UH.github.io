  4.1메서드

예제 String 객체 변수 str = “Hello”

.substring(시작위치, 종료위치)

substring(시작위치,종료위치) 메서드는 string을 배열로 생각했을 때 시작위치에서 종료위치 바로 앞 단어까지를 추출해 String 객체로 반환하는 메서드이다.

.charAt(위치)

charAt(위치) 메서드는 String을 배열로 생각했을 때 위치에 해당되는 인덱스의 한 문자를 char타입으로 반환하는 메서드이다.


.equals(문자열) & .equalsIgnoreCase(문자열)

.equals(문자열)과 .equalsIgnoreCase(문자열)은 문자열과 비교하여 같은지 다른지 비교하여 Boolean타입으로 값을 리턴하며 두 차이는 전자는 소대문자를 구분하며 후자는 소대문자를 무시하고 비교한다.

  • ==를 쓰지 않는 이유는 Value를 비교하는 것이 아는 주소를 비교하기 때문이다. 예를 들어 str은 “Hello”를 가르키는 포인터의 역할로 주소 값을 가지고 있다. 그리고 str이 가르키는 객체는 “Hello”인데 str1이라는 변수도 “Hello” 라면 두 변수가 가르키는 객체가 동일 주소에 저장된 “Hello” 객체라면 True겠지만 다른 주소에 각각 달리 저장된 “Hello”들이라면 False가 return된다.
